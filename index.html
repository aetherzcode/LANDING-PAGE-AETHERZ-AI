Mulai Chat

Revolusi Chatbot AI

Pengalaman chat interaktif yang cerdas, responsif, dan personal

Fitur

  • Setiap user/pengguna memiliki data chat masing masing
  • Login hanya membutuhkan nama
  • Refresh halaman tidak akan mereset riwayat chat pengguna
  • Memiliki fitur untuk menghapus riwayat chat obrolan
  • Chat interaktif dengan AI
  • Dukungan untuk bahasa Indonesia dan Inggris
  • Kemampuan Text-to-Speech (TTS)
  • Mode gelap/terang
  • Kemampuan untuk menyalin kode pemograman yang dihasilkan oleh AI

Teknologi yang Digunakan

  • Node.js
  • Express.js
  • EJS (Embedded JavaScript templating)
  • Axios
  • Howler.js
  • Font Awesome
  • Highlight.js
  • MongoDB
  • TTS (Text To Speech)
